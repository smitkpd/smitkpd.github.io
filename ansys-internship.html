Ansys: Meshing Intern

(Aug - Dec 2023)

One of my first projects at Ansys was to automate testing of the Axisymmetric meshing algorithm to make the time-consuming process of manually checking mesh settings significantly more efficient. Specifically, I wrote a Python script which detects mesh nodes, computes how well they conform to the geometry's circumference, and modifies Sweep number of divisions accordingly. This saved time by 10+ hours for testing 1000+ CAD files, and improved test quality by ensuring the mesh was optimized for each geometry (by checking mesh metrics like aspect ratio).
