Ansys: Mesh Testing Intern

(Aug - Dec 2023)

Automatated testing of Axisymmetric Meshing

As one of my first projects at Ansys, I automated testing of a detection feature for 1400+ Axisymmetric bodies, as well as evaluation of Axisymmetric meshing. Specifically, I wrote a Python script which leverages Ansys Mechanical APIs to import geometry, generate mesh and validate mesh element count. I reduced test runtime from 30+ hours to 5 hours, enabling the Development team to rapidly improve their feature.

Additionally, some of the tests required manual intervention to improve the mesh for certain bodies. I implemented an algorithm which checks how well nodes conform to a geometry's circumference, and automatically improves the mesh, increasing test success rate by 39%

Migration of JavaScript mesh regression tests to Python

There are thousands of regression tests which ensure stability of mesh algorithms as the source code is edited. I developed a Python workflow to compile 100+ JavaScript test files per mesh algorithm into a single Python file to enable faster debugging and more robust mesh validation. I rigorously documented my approach and scripts to enable my team to build upon my work for migrating other mesh tests to Python.

Key Takeaways:-

  • • Regular expressions to extract test commands and target mesh metrics from JavaScript and XML files

  • • Python dictionaries to organize and map commands to Ansys Mechanical APIs

  • • Evaluating node and element placement for contact regions and shell mesh
