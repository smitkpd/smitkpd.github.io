Ansys: Mesh Testing Intern

(Aug - Dec 2023)

Automatated testing of Axisymmetric Meshing

As one of my first projects at Ansys, I automated testing of a detection feature for 1400+ Axisymmetric bodies, as well as evaluation of Axisymmetric meshing. Specifically, I wrote a Python script which leverages Mechanical APIs to import geometry, generate and validate mesh for various geometries. I reduced test runtime from 30+ hours to 5 hours, enabling the Development team to rapidly improve their feature. Additionally, some of the tests required manual intervention to improve the mesh for certain bodies. I implemented an algorithm which checks how well nodes conform to a geometry's circumference, and automatically improves the mesh, increasing test success rate by 39%.

Migration of JavaScript mesh regression tests to Python

There are thousands of regression tests which ensure stability of mesh algorithms after changes to the source code. I developed a Python workflow to compile 100+ JavaScript test files per mesh algorithm into a single Python to enable faster debugging and more robust mesh validation. First, I parsed through JavaScript and XML files to retrieve test commands and target mesh metrics, then mapped them to Ansys Mechanical APIs using Python dictionaries, leveraging APIs in Ansys Mechanical to automate the following process: geometry import, face/edge selections, mesh method application and mesh evaluation (element quality, mesh volume tolerance, etc.) I learned how to apply regular expressions, use Python dictionaries and navigate file dependencies to reduce test runtime, and create an efficient automated workflow with Python.
