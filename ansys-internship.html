Ansys: Mesh Testing Intern

(Aug - Dec 2023)

Axisymmetric meshing automation

As one of my first projects at Ansys, I automated testing of the Axisymmetric Sweep meshing algorithm to reduce time associated with evaluating regression tests manually. Specifically, I wrote a Python script which detects mesh nodes, computes how well they conform to a geometry's circumference, and adjusts Sweep number of divisions accordingly. This saved time by 30+ hours for testing 1000+ CAD files. Moreover, it ensured the mesh was optimized for each geometry, thus increasing the success rate of mesh tests, which would have otherwise failed with default mesh settings.

Regression test automation

Currently, I am driving the migration of all JavaScript mesh test files to Python, for reducing total number of regression tests. I am acheiving this by leveraging APIs in Ansys Mechanical to automate the following process: geometry import, face/edge selections, mesh method application and mesh evaluation (element quality, mesh volume tolerance, etc.) I learned how to apply regular expressions, use Python dictionaries and navigate file dependencies to reduce test runtime, and create an efficient automated workflow with Python.
